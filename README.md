# irish-tourism-prediction-ml

##  Project Overview
This project analyzes Irish tourism datasets to identify visitor trends and predict tourism spending and stay patterns using regression models.

##  Technologies Used
- Python
- Pandas & NumPy
- Scikit-Learn
- Matplotlib & Seaborn

##  Approach
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Model development using:
  - Linear Regression
  - Random Forest Regression
  - Support Vector Regression
- Model evaluation using R² and MSE metrics

##  Results
The Random Forest model achieved the highest predictive accuracy, demonstrating strong performance in forecasting visitor spending patterns.

##  Key Skills Demonstrated
Data cleaning, statistical analysis, regression modeling, model evaluation, data visualization.
